# {task_id} 阻塞点记录

> 记录任务执行中遇到的所有阻塞点。
>
> **来源分工**：
> - `[Hook-auto]` 由 `blocker-tracker.py` 自动写入（工具层错误：编译/测试/权限/网络失败）
> - `[Agent主动]` 由 Agent 主动写入（业务层阻塞：需求不清/契约歧义/技术决策无据）
>
> **格式**：每条用 `---` 分隔，保留时间戳顺序。

## Blocker 类型说明

| 大类 | 子类 | 来源 | 示例 |
|------|------|------|------|
| 环境 | 编译/测试/权限/网络 | Hook 自动 | mvn compile 失败 |
| 执行 | 测试/契约不匹配 | Hook 自动 | 契约测试 assertion 失败 |
| 信息 | 需求缺失/契约歧义/技术决策/外部依赖 | Agent 主动 | AC 描述不具体 |
| 流程 | 步骤缺失/顺序错误 | Agent 主动 | 契约未冻结就开工 |
