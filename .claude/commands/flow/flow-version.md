# flow-version

查看和切换 Flow 版本。

## 使用场景

- 查看所有可用的 Flow 版本
- 切换到旧版本（如 v2.3）
- 回滚有问题的更新

## 命令

### 查看版本列表

```
/flow-version
```

## 输出示例

```

当前版本: v2.4
当前分支: master

v2.4 ← HEAD
v2.3
v2.2
v2.1
...

使用 /flow-pull --version <版本> 切换
```

## 版本切换

版本切换使用 `/flow-pull` 命令：

```
/flow-pull --version v2.3
```

## 版本说明

| 版本 | 说明 |
|------|------|
| v2.4 | AI 反馈闭环（自审+洞察+进化） |
| v2.3 | 长程任务自动执行 |
| v2.2 | 删除 orchestrator Agent |
| v2.1 | 事件驱动架构 + TAPD 同步 |

## 注意事项

- 切换版本会改变 Flow 的功能
- 建议在测试环境中先验证
- 切换后需要重启 session
